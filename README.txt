Desktop client for WebCT

This application is intended to support desktop access to files and other resources in WebCT, as deployed at the University of Nottingham.

Copyright (C) 2011, The University of Nottingham
Written by Chris Greenhalgh.

In is currently a prototype and concept demonstration. It interacts with WebCT via WebCT's web service interface (in particular the Context and Files APIs) and via the web browser interface (emulating aspects of a browser session).

It can be currently used to download files and some other content from WebCT courses.
